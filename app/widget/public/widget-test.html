2. Integration Code

To add this widget to your site, copy and paste this code:

<!-- Add this to your HTML's <head> -->
<link href="https://YOUR-DOMAIN.com/chat-widget.css" rel="stylesheet" />

<!-- Add this where you want the chat widget to appear -->
<div id="chat-widget" data-agent-id="YOUR_AGENT_ID"></div>

<!-- Add this right before the closing </body> tag -->
<script src="https://YOUR-DOMAIN.com/chat-widget.js" defer></script>

3. Customization

You can customize the widget's appearance using these CSS variables:

:root {
  --chat-header-bg: #3B82F6;  /* Header background */
  --chat-accent: #00D4FF;     /* Accent color for buttons */
  --chat-bg: #FFFFFF;         /* Main background */
  --chat-text: #1f2937;       /* Main text color */
  --chat-text-light: #6b7280; /* Secondary text */
  --chat-border: #e5e7eb;     /* Borders */
  --chat-shadow: 0 4px 6px -1px rgba(0,0,0,.1); /* Shadow */
}
